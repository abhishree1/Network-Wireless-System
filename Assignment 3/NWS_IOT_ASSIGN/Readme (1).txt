Steps to be followed for successful execution of the code:

1. Start the MQTT server
- Go into the folder where MQTT is installed.
- Inside bin folder you will find run.bat/ run.sh file.
- if you are using windows run the run.bat file by double clicking on it.
- if you are using linux run the run.sh file in terminal using ./run.sh command.

2. Run the run_game_* file provided in the folder
- For windows run run_game_windows.ps1 file in powershell.
- For linux run run_game_linux.sh file in terminal.
